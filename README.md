# Native Speech App

An AI-powered web app that rewrites English sentences the way a native American speaker would naturally say them.

The app also applies emotional tone and visual speech melody using uppercase and lowercase patterns to simulate native prosody.

## Features

- Natural native phrasing
- Emotional tone selection
- Speech melody visualization
- AI-generated responses
- Flask backend
- HTML/CSS frontend

## Technologies Used

- Python
- Flask
- Groq API
- HTML
- CSS

## Example

Input:
I am opening the door

Emotion:
Angry

Output:
I'M opening the DOOR, okay?

## Installation

Clone the repository:

```bash
git clone https://github.com/Hl20012018/native-speech-app.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
API_KEY=your_api_key_here
```

Run the app:

```bash
python app.py
```

## Environment Variables

Create a `.env` file in the root directory:

```env
API_KEY=your_api_key_here
```
